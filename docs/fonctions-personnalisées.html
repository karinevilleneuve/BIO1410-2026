Chapitre 7 9. Fonctions personnalisées

Une fonction permet d’exécuter un bloc de code réutilisable.

Code 
carre <- function(n) {
  return(n^2)  # Fonction qui renvoie le carré d'un nombre
}
print(carre(4))  # Appel de la fonction
## [1] 16
